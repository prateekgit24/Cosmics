# COSMICS

**Cosmics** provides high-quality software resources, premium UI kits, and full-stack boilerplates tailored for modern developers. 

Welcome to the official repository for the Cosmics website. This project is a highly optimized, fully static HTML/CSS/JS frontend designed for maximum performance, excellent SEO, and a premium user experience featuring a sleek dark mode aesthetic.

## 🚀 Features
- **Curated Learning Center**: Structured guides and paths for frontend, backend, and startup engineering.
- **Premium Resources**: Handpicked interactive courses, open-source APIs, UI generation tools, and cloud services.
- **Dark-Themed UI**: A completely custom, responsive, and carefully crafted design system using modern CSS properties.
- **High Performance**: 100% static architecture, ensuring instant page loads without the overhead of heavy JavaScript frameworks.
- **Built-in SEO**: Fully optimized with dynamic sitemaps, robots.txt, semantic HTML, and comprehensive Open Graph mapping for flawless social sharing.
- **Integrated Web3Forms**: Contact and application forms routed securely without needing a custom backend server.

## 🛠️ Technology Stack
- **HTML5**: Semantic and structural integrity.
- **Vanilla CSS3**: Leveraging CSS variables (`:root`) for global theming, Flexbox/Grid for layout, and smooth transitions/micro-animations.
- **Vanilla JavaScript**: Lightweight interactions (like custom form submission handling).
- **Google AdSense Integration**: Properly configured Auto Ads.

## 📁 File Structure
- `index.html`: The main landing page.
- `resources.html`: The comprehensive directory of developer tools and courses.
- `learning.html`: Educational hubs and tutorials.
- `career.html`: Internship application portal (integrated with Web3Forms).
- `purchase.html`: Premium UI kits and boilerplates catalog.
- `about-us.html` / `contact-us.html`: Company information and communication channels.
- `privacy-policy.html` / `terms-conditions.html`: Legal documentation.
- `sitemap.xml` / `robots.txt`: Search engine crawling optimization.
- `404.html`: Custom error routing.
